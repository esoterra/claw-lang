# Integers

Signed and unsigned integers of sizes 8, 16, 32, and 64 bits are available.

| Size | Signed | Unsigned |
|-|-|-|
| 8 bits | `s8` | `u8`
| 16 bits | `s16` | `u16`
| 32 bits | `s32` | `u32`
| 64 bits | `s64` | `u64`

Floating point numbers are available in 32 and 64 bit sizes.

| Size | Name |
|-|-|
| 32 bits | `f32`
| 64 bits | `f64`

## Constructing

Decimal number literals

```rust
let a: u32 = 10
```
